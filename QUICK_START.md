## üéØ Quick Start (3 Steps)

### Step 1: Install Laragon
- Download: https://laragon.org/download/
- Install to `C:\laragon`

### Step 2: Copy Files
- Copy this folder to `C:\laragon\www\`

### Step 3: Install MyAcademy
- Double-click `MYACADEMY.bat`
- Select option **[1] Install System**
- Wait 5-10 minutes
- Done!

## üìñ Menu Options Explained

### [1] Install System
- **When:** First time only
- **What:** Sets up everything automatically
- **Note:** Requires Administrator rights

### [2] Start MyAcademy Server ‚≠ê
- **When:** Every morning
- **What:** Starts the system
- **Note:** Keep window open while using

### [3] Stop MyAcademy Server
- **When:** End of day
- **What:** Stops the system safely

### [4] Create Backup ‚≠ê
- **When:** Every Friday
- **What:** Creates full backup
- **Note:** Copy to external drive!

### [5] Update System
- **When:** New version available
- **What:** Updates to latest version
- **Note:** Backup first!

### [6] Show Network Info
- **When:** Sharing with teachers
- **What:** Shows teacher access URL

### [7] Check System Status
- **When:** Troubleshooting
- **What:** Diagnoses problems

### [8] Exit
- **When:** Done using menu
- **What:** Closes the program

## üíº Daily Workflow

### Morning
1. Double-click `MYACADEMY.bat`
2. Select **[2] Start Server**
3. Share URL with teachers (option 6)

### Evening
1. Double-click `MYACADEMY.bat`
2. Select **[3] Stop Server**

### Friday
1. Double-click `MYACADEMY.bat`
2. Select **[4] Create Backup**
3. Copy backup to external drive
