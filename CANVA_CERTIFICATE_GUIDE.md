### Step 1: Create New Design
1. Go to **Canva.com** (free account)
2. Click **Create a design**
3. Select **Custom size**
4. Enter: **1754 x 1240 pixels** (A4 landscape)
5. Click **Create new design**

### Step 4: Export Template

1. Click **Share** (top right)
2. Select **Download**
3. Choose **PNG** format
4. Click **Download**
5. Save as: `achievement.png`, `completion.png`, or `excellence.png`

### Step 5: Upload to MyAcademy

## Template Types

### 1. Achievement Certificate
**Use for:** Academic excellence, sports, competitions
**Colors:** Gold and navy blue
**Style:** Formal, prestigious
**File:** `achievement.png`

### 2. Completion Certificate
**Use for:** Course completion, program graduation
**Colors:** Green and teal
**Style:** Professional, clean
**File:** `completion.png`

### 3. Excellence Certificate
**Use for:** Top performers, special recognition
**Colors:** Purple and gold
**Style:** Elegant, premium
**File:** `excellence.png`

## Text Positioning Guide

The system will add text at these positions:

| Element | Position | Font Size | Color |
|---------|----------|-----------|-------|
| Title | Top center (380px from top) | 32px | #2C5282 |
| Student Name | Center (520px from top) | 48px | #1A1A1A |
| Description | Below name (620px from top) | 20px | #4A5568 |
| Date | Bottom (720px from top) | 18px | #4A5568 |

**Leave 200px margins** on all sides for text overlay.

## Troubleshooting

**Problem:** Text doesn't fit
**Solution:** Use shorter titles or adjust font size in settings

**Problem:** Template looks blurry
**Solution:** Export at higher quality (PNG, not JPG)

**Problem:** Colors don't match school
**Solution:** Use school brand colors in Canva color picker

**Problem:** Upload fails
**Solution:** Ensure file is PNG, under 10MB, 1754x1240px
